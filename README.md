# custom-plugins

## 目录结构说明

```
├── features
├── plugins
│   ├── org.ruyisdk.core
│   ├── org.ruyisdk.devices
│   ├── org.ruyisdk.news
│   ├── org.ruyisdk.packages
│   ├── org.ruyisdk.intro
│   ├── org.ruyisdk.ruyi
│   └── org.ruyisdk.ui
└── README.md

```

* org.ruyisdk.core:基础类，供其它插件调用的公共部分
* org.ruyisdk.devices:RISC-V设备管理
* org.ruyisdk.packages:包资源管理器
* org.ruyisdk.intro:定制化欢迎界面
